# python-flask-app for server check

![Apache 2.0 License](https://img.shields.io/badge/License-Apache2.0-orange)
![Python](https://img.shields.io/badge/Built_with-Python-blue)
![Pytest](https://img.shields.io/badge/Powered_by-Pytest-green)

Python Flask App to check that a server is alive and running.

## Run it:
```bash
flask run -h localhost -p 3000
```
or
```bash
nohup flask run -h localhost -p 3000 &
```
